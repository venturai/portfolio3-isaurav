HTML is a standardized system for tagging
text files to achieve font, color, graphic
and hyper link effects on web pages.
CSS is a style sheet language that is used for
describing the look and formating of a document
written in a markup language.
Bootstrap is a technique of loading a program into
a computer by means off a few initial instructions
that enable the introduction of the rest of the program.
JavaScript is an object-oriented computer programming language
commonly used to create interactive effects within web browsers. jQuery is a fast, small, and feature-rich JavaScript library.
It makes things like HTML documents traversal and manipulation, event handling
animation.
